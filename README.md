# Student Registration Form

## Overview

This project is a simple Student Registration Form created using HTML.

The form collects basic student information such as name, age, gender, skills. It is designed to practice HTML form elements and table creation.

## Features

* Student Name input
* Age input
* Gender selection using radio buttons
* Course selection using dropdown menu
* Contact information fields
* Address field using textarea
* Submit button
* Student details displayed in a table

## Technologies Used

* HTML5

## Concepts Practiced

* HTML Forms
* Labels
* Input Fields
* Radio Buttons
* Dropdown Menus
* Textarea
* Buttons
* Tables
* Basic Page Structure

## Learning Outcome

Through this project, I learned:

* How to create forms in HTML
* How to collect user input using different form elements
* How radio buttons work using the same name attribute
* How to create and organize tables
* How to structure a simple web page using semantic HTML
